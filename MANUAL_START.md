# Инструкция по ручному запуску сайта

Следуйте этим шагам, чтобы запустить сайт на своем компьютере.

## 1. Предварительные требования
Убедитесь, что у вас установлен **Node.js**.
Проверить можно командой в терминале:
```bash
node -v
```
Если команда не найдена, скачайте и установите Node.js с [официального сайта](https://nodejs.org/).

## 2. Открытие терминала
Откройте терминал (Terminal на MacOS или PowerShell/Cmd на Windows) и перейдите в папку с проектом.
Пример команды (замените путь на ваш):
```bash
cd "/Users/dirtyglock/Desktop/test main"
```

## 3. Установка зависимостей
Если вы запускаете проект впервые или были изменения в библиотеках, выполните:
```bash
npm install
```
*Примечание: Если возникнут ошибки с версиями (например, ERESOLVE), попробуйте `npm install --legacy-peer-deps` или `npm install --force`.*

## 4. Запуск в режиме разработки
Для запуска локального сервера выполните:
```bash
npm run dev
```

После успешного запуска вы увидите сообщение:
```
- Local: http://localhost:3000
```
Откройте [http://localhost:3000](http://localhost:3000) в вашем браузере.

## 5. Запуск готовой версии (Production)
Если вам нужно проверить, как сайт будет работать "в бою":

1. Соберите проект:
   ```bash
   npm run build
   ```
2. Запустите собранную версию:
   ```bash
   npm start
   ```

## Возможные проблемы
- **Ошибка "Address already in use"**: Значит порт 3000 занят. Остановите другие процессы или перезагрузите компьютер.
- **Ошибки при `npm install`**: Попробуйте удалить папку `node_modules` и файл `package-lock.json`, затем запустите `npm install` заново.
